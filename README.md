# Ramzy Albazel — AI & Full Stack Developer Portfolio

Professional portfolio for Ramzy Albazel, showcasing expertise in AI Engineering and Full Stack Development.

## Technologies Used
- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI
- **Backend**: Supabase (Database & Auth), Edge Functions
- **AI**: Google Gemini (Integration)

## Deployment on Vercel

1. Connect your GitHub repository to Vercel.
2. Add the following environment variables in Vercel project settings:
   - `VITE_SUPABASE_URL`: Your Supabase project URL.
   - `VITE_SUPABASE_ANON_KEY`: Your Supabase anonymous key.
3. Deploy!

## Local Development

```bash
npm install
npm run dev
```
